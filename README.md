# AI Transfer IQ ⚽

AI Transfer IQ is a Machine Learning based football transfer value prediction system built using FIFA dataset. The project predicts player transfer values using Random Forest Regression and displays results through a Flask web application.

## Features
- Machine Learning prediction model
- Transfer value estimation
- Flask web interface
- Currency conversion to INR
- Clean UI design

## Technologies Used
Python  
Pandas  
Scikit-learn  
Flask  
HTML/CSS  

## Project Structure

Backend
- app.py
- train_model.py
- fifa.csv

templates
- index.html

requirements.txt

## How to Run

Install dependencies:
pip install -r requirements.txt

Train model:
python train_model.py

Run project:
python app.py

Open:
http://127.0.0.1:5000

## Note
model.pkl is not uploaded due to GitHub file size limits. Run train_model.py to generate it.
